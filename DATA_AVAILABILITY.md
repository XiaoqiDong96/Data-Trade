# Data Availability

This GitHub repository does not contain the RapidAPI platform data used in the empirical analysis.

Excluded files include:

- Raw GraphQL and HTML responses.
- Normalized platform CSV tables.
- Merged research CSV tables.
- Model input CSV files.
- Data handoff zip archives.
- Literature PDF files.

The repository contains code, documentation, article outputs, figures, and Markdown result tables. Collaborators with access to the private data package should place the files in the local paths documented in `README.md` and `rapidapi_crawl/docs/rapidapi_collaborator_full_data_manifest.md`.

The merged data package used for collaboration was generated locally as:

```text
handoff/rapidapi_merged_tables_20260618.zip
```

That archive is intentionally not tracked in git.
